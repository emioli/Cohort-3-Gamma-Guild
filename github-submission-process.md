# **GitHub Submission Process**

## **Step-by-Step Instructions:**

1. **Fork the main class repository** to your GitHub account (if you haven't already)

2. **Clone the forked repository locally**
   ```bash
   git clone https://github.com/your-username/Web3-Class-Repository.git
   cd Web3-Class-Repository
   ```

3. **Create a new branch for Week 1**
   ```bash
   git checkout -b week-1-assignment
   ```

4. **Create your folder structure**
   ```bash
   mkdir -p submissions/week-1/your-name/day-1
   ```

5. **Create and complete your assignment file**
   - File name: `your-name-blockchain-infrastructure.md`
   - Location: `submissions/week-1/your-name/day-1/`

6. **Add your completed assignment**
   ```bash
   git add .
   ```

7. **Commit your changes**
   ```bash
   git commit -m "chore: submit week 1 day 1 blockchain research assignment"
   ```

8. **Push to YOUR forked repository**
   ```bash
   git push origin week-1-assignment
   ```

9. **Create a Pull Request to the main class repository**
   - Go to the original class repository on GitHub
   - Click "Pull Requests" â†’ "New Pull Request"
   - Click "compare across forks"
   - Select your fork and `week-1-assignment` branch
   - **PR Title:** `Week 1 Day 1 - [Your Full Name]`
   - **PR Description:** Brief summary of what you researched
   - Submit the PR
